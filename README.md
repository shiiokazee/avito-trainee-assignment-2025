## Инструкция по запуску

Выполните следующие команды:

- `yarn`/`npm init` - для установки зависимостей
- Скопируйте файл `.env.example` в `.env` и укажите все необходимые для работы приложения переменные
- Выполните `yarn dev`/`npm start dev` для запуска в dev-режиме

Дополнительные скрипты описаны в разделе "[Скрипты](#скрипты)"

## Обоснование выбора технологий

- **Vite** - быстрый, гибкий и при этом не требующий конфигурации сборщик, хорошая поддержка HMR
- **MUI** - крайне популярная библиотека компонентов. Поддерживает стилизацию, очень гибкая и настраиваемая и содержит все необходимые базовые компоненты для выполнения проекта. Также имеет множество расширений, в том числе `@mui/x-charts`, которое было выбрано для отрисовки графиков
- **react-query** - была выбрана из-за подержки кэширования "из коробки" и удобного API
- **swagger-typescript-api** - для кодогенерации на основе OpenAPI-схемы, чтобы не прихоидилось описывать контракты и методы вручную

## Скрипты

- `yarn codegen -p <путь_до_схемы>` - генерация api-методов и ts-типов на основе схемы OpenAPI
- `yarn build` - для создания production сборки проекта
- `yarn lint` - линтинг при помощи eslint
- `yarn typecheck` - проверка корретконсти типов
